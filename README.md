# FinTrack — דשבורד פיננסי חכם

דשבורד לניהול פיננסי אישי עם ניתוח הוצאות, השקעות, משכנתא ותובנות חכמות.

## מבנה הפרויקט

```
fintrack-project/
├── index.html      ← מבנה הדף (HTML בלבד)
├── styles.css      ← כל עיצוב המערכת
├── data.js         ← נתוני עסקאות והכנסות (יוחלף ב-API בייצור)
├── app.js          ← לוגיקת האפליקציה (charts, filters, tabs)
└── auth.js         ← מערכת הרשמה, כניסה, תנאי שימוש, פרופיל
```

## הפעלה מקומית

פתח את `index.html` בדפדפן. לא נדרש שרת.

## טכנולוגיות

- Chart.js 4.4.1 — גרפים
- SheetJS 0.18.5 — קריאת קבצי Excel
- LocalStorage — שמירת נתוני משתמש
- RTL Hebrew UI
